
# Awesome-Obsidian-zh

![](https://img.shields.io/badge/language-Chinese-white.svg) ![](https://img.shields.io/github/stars/PKM-er/awesome-obsidian-zh.svg)

## 简介

- 🀄本项目旨在持续汇集 Github 上的 [Obsidian](https://obsidian.md/) **原生中文插件**和**自带中文翻译**的流行插件，为中文用户提供一个发现、安装和参与贡献的中心。

## 官方中文社区与资源

- **[Obsidian 中文论坛](https://forum-zh.obsidian.md/)**: 官方的中文用户交流社区。
- **[Discord 官方中文频道](https://discord.com/channels/686053708261228577/735629542906920993)**: 实时的在线聊天频道。

## 社区汉化指引

**无需代码背景，你也可以为社区做贡献！**

许多热门的国际插件都开放了社区翻译。通过简单的步骤，你就可以帮助它们更新中文语言文件，让更多中文用户受益。

- **通用教程**：
    1. 在插件的 GitHub 仓库中找到语言文件目录（通常名为 `lang`, `locale`, `i18n`[^1], `l10n`[^2]）。
    2. 找到中文文件（如 `zh.json` 或 `zh-cn.ts`）。
    3. 参考同目录下的英文文件 (`en.json`)，将未翻译的或翻译不佳的文本进行修改。
    4. 通过 `Pull Request` (PR) 将你的修改提交给原作者。
    5. 等待作者审核合并。
- **译名参考**: 为了保持一致性，请参考 [Obsidian 官方翻译库中的常用译名](https://github.com/obsidianmd/obsidian-translations/blob/master/zh.json)。

## 贡献指南

我们欢迎所有用户参与共建，让这个列表变得更好！

如果你发现或开发了优秀的中文插件/主题，或希望对现有条目进行补充修改，请通过 **[提交一个 Issue](https://github.com/PKM-er/awesome-obsidian-zh/issues)** 或 **[发起 Pull Request](https://github.com/PKM-er/awesome-obsidian-zh/pulls)** 的方式告知我们。

**提交标准:**

1. **强相关性**: 插件/主题由中文开发者主导，或对中文环境有特别优化，或拥有高质量的中文翻译。
2. **可用性**: 插件/主题在 Obsidian 最新版中可稳定运行，且有一定数量的用户。
3. **遵循格式**: 提交时请参考本文档的表格格式。
